# Description
Using the open source dataset provided by the CAVIAR project to apply the skills from the "Python for Data Analysis" course.

# Data
This project uses the CAVIAR dataset "Clips from Shopping Center in Portugal (2nd Set)" front view.
The XML ground truth files can be found here: https://groups.inf.ed.ac.uk/vision/DATASETS/CAVIAR/CAVIARDATA1/
Download all 26 XML files from the 2nd Dataset but only front view!
Place the donloaded folders inside a 'data\' directory in the root of this repository.
